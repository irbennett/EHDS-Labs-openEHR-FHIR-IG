# FHIR Connect Mappings

openEHR FHIR Connect mappings define the transformation between FHIR resources and openEHR compositions,
enabling bidirectional data exchange between FHIR-native systems and openEHR Clinical Data Repositories.

Mapping files are placed in `input/includes/openehr-input/fhir-connect/`.
