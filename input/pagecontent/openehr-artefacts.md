# openEHR Artefacts

This section covers the openEHR-specific artefacts in this Implementation Guide.

## Templates

openEHR templates constrain archetypes to define the specific data sets used in this IG.
See the [Templates](openehr-templates.html) page for full details.

## Composition Examples

openEHR composition instances showing how clinical data is structured according to openEHR archetypes and templates.
Each example is available in one or more serialisation formats (FLAT, STRUCTURED, or CANONICAL JSON).
See the [Composition Examples](openehr-compositions.html) page for full details.

## AQL Queries

openEHR Archetype Query Language (AQL) queries for retrieving data from an openEHR Clinical Data Repository.
